DEVOPS ASSIGNMENT 4

This is a simple python application that serves as a 2-number calculator for basic operations, 
Featuring a simple command line UI, and shouldn't crash due to incorrect user input.
It also features a bonus rock, paper, scissors game against the computer!

to run the program locally:
python3 Devops_Assignment4.py
no other dependencies are required.

to run through Docker:
docker pull moul0087/devops_assignment4
docker run -it --rm devops_asssignment4
